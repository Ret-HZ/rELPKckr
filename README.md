# rELPKckr
An **ELPK** archive extractor/packer used in games by ***syn Sophia***.
<br />
<br />

### Usage
```cmd
rELPKckr <file or directory>
```
Supports drag & drop
<br />
<br />

### Supported games
* Kurohyou: Ryu Ga Gotoku Shinsho 
* Kurohyou 2: Ryu Ga Gotoku Ashura Hen
